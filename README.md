# ⚽ نادي الاسطورة - Legend Club

لعبة كرة قدم أونلاين شاملة للهواتف المحمولة - أنشئ ناديك وقده نحو المجد!

## 🎮 مميزات اللعبة

- **إنشاء الأندية**: اختر اسم ناديك، شعاره، ألوانه، وتشكيلته (8 تشكيلات مختلفة)
- **إدارة الفريق**: 22 لاعب بتفاصيل كاملة (سرعة، تسديد، تمرير، مراوغة، دفاع، قوة بدنية)
- **محاكاة المباريات**: نظام محاكاة واقعي مع أحداث مباشرة
- **4 بطولات**: دوري الماس ⭐، دوري الذهب 🥇، دوري الفضة 🥈، كأس البرونز 🥉
- **سوق الانتقالات**: اشترِ وبع اللاعبين في سوق مفتوح
- **نظام الحزم**: حزم برونزية، فضية، ذهبية، وأسطورية
- **التدريب**: حسّن مهارات لاعبيك بالتدريب اليومي
- **الإنجازات**: 12 إنجاز مختلف مع مكافآت قيّمة
- **المواسم**: نظام مواسم مع تصنيفات ومكافآت
- **المكافآت اليومية**: هدايا يومية ومكافآت مشاهدة الإعلانات
- **لوحة المتصدرين**: تنافس مع اللاعبين عالمياً
- **لوحة تحكم المسؤول**: إدارة كاملة للمستخدمين واللاعبين والبطولات

## 🛠️ التقنيات المستخدمة

- **Frontend**: Next.js 16 + React 19 + TypeScript + Tailwind CSS 4
- **Backend**: Next.js API Routes (Serverless)
- **Database**: PostgreSQL (Supabase)
- **ORM**: Prisma 6
- **State Management**: Zustand
- **UI Components**: shadcn/ui + Radix UI
- **Animations**: Framer Motion
- **Mobile APK**: Capacitor 6

## 🚀 التشغيل المحلي

```bash
# تثبيت المكتبات
npm install

# إنشاء Prisma Client
npx prisma generate

# تشغيل الخادم المحلي
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في متصفحك.
